Questions
Frequently Asked
What technology stack is MediCore built on?
MediCore is built with Python Django for the backend, Bootstrap 5 for the frontend, and Chart.js for analytics. The default database is SQLite (easily upgradeable to PostgreSQL). It runs on any server that supports Python 3.10+.
Do I get the full source code?
Yes — you receive 100% of the source code with no obfuscation or licensing locks. You can modify, extend, and deploy it however you need for your hospital.
How difficult is the setup?
If you have basic Python knowledge, setup takes under 30 minutes. A detailed step-by-step installation guide is included. For Custom plan clients, we handle the deployment for you.
Can I customise it for my hospital's branding?
Absolutely. The system is designed with clean, modular templates. You can change the hospital name, logo, colors, and add your own departments and staff with no major code changes.
Is there a monthly subscription fee?
No. MediCore is a one-time purchase. You own the software permanently. There are no recurring fees, no SaaS charges, and no per-user costs.
What kind of support do I get after purchase?
Pro plan includes 3 months of email support for setup and bug queries. Custom plan includes 6 months of priority support plus deployment assistance. Basic plan includes documentation only.
